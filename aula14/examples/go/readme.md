# AP1 em GO

Neste repositório encontram-se os exemplos feitos em go
dos comandos vistos em sala de aula.

## Compilação em Go

Para compilar e executar um arquivo, siga o seguinte comando:
```go run <nome-do-arquivo.go>```

Para somente compilar o programa, siga o seguinte comando:
```go build <nome-do-arquivo.go>```
